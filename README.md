# Madhayana Market Complete v7

Fitur utama:
- Warna lebih cerah, familiar marketplace, dan user friendly.
- Slider iklan/highlight otomatis dan bisa diklik direct.
- Filter produk: all, harga, produk/kategori, jenis, nama/search, rating/sort.
- User checkout QRIS, invoice, Firestore orders, login Google.
- Operator: tambah, edit, duplikat, hapus produk.
- Produk: nama, gambar, deskripsi, harga, rating, jenis, kategori, stok, tags.
- Komentar: tambah, balas contoh, pin, hapus, hide, blokir, report pilihan.
- Tata letak web: theme, jumlah kolom, rounded, banner.
- Privacy dan Terms sudah tersedia.

Cara jalan lokal:
npm install
npm run dev

Deploy Vercel:
Build Command: vite build
Output Directory: dist
Install Command: kosong/default
